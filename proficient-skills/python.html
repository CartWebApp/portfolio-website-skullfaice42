What I Learned

Python was the first programming language I coded in. I followed a few tutorials in the beginning to help me learn, but eventually, I started learning through AI.

In this example, I'm coding a password scrambler. The purpose is to generate a password from a regular word. It takes a word like "House" and turns it into something like #()uS3, |-|OU5E, or [-]{}U$3. This is useful if you want your password to be based on a word you know, but you don't want something randomly generated by Google.
